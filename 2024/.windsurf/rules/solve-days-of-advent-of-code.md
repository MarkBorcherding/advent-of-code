When solving an advent of code problem always copy the input and instructions for part 1 and 2 into files. extract the sample problem data from the instructions and save that in it's own file. Implement part 1 and test it agsinst the sample. If the answer matches, run part 1 against the real input data and submit that as the answer for part one. Use the feed back to decide if adjustments are needed or you can move on to part 2. Do the same thing by saving the part 2 instuctions to its own file, impklementing the solution, testing against the sample answer and submitting it as the real answer if things match. Adjust the implementation if answers don't match and resubmit.
